# Personal Website

Adi Desai's password-protected portfolio website showcasing research, projects, awards, and photography.

**Website**: [adidesai.org](https://adidesai.org)

## Overview

A comprehensive portfolio for a high-achieving Lakeside School junior, highlighting work in molecular biology, computational biology, biosensor engineering, and community health initiatives.

## Technologies

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Playfair Display, Source Sans 3)
- SHA-256 password protection via Web Crypto API
- Cloudinary CDN for images
- GitHub Pages hosting

## Main Pages

| Page | Description |
|------|-------------|
| `login.html` | Password-protected entry |
| `index.html` | About me, education, research experience |
| `projects.html` | 7 major project showcases |
| `interests.html` | Photography, languages, music, crew |
| `awards.html` | Awards and publications |
| `photos.html` | Photography portfolio with lightbox |
| `blog.html` | 25+ blog posts on diseases and medical tech |
